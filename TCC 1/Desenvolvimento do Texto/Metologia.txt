%% JUSTIFICATIVA DO SUBSISTEMA DAS MATRIZES

Como no primeiro momento não está preocupado com a posição linear do pêndulo, é possível realizar um subsistema das matrizes A,B e C. Dessa forma, esse novo sistema
não perderá rank quando for testado a observabilidade, já que não há mais o estado x. O novo subsistema é visto abaixo:

A = 3x3    B = 3x1   C = 1x3   variáveis = xdot, theta, thetadot







%% Filtro de Kalman

xdot = Ax + Bu + V_d * d + 0*n
y    = Cx + Du + 0*d + V_n * n
